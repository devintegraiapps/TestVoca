# Deploy rápido via GitHub Pages (/docs)

1) Suba a pasta `docs/` para o branch `main` deste repositório.
2) Em Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main / /docs
3) Abra: https://devintegraiapps.github.io/TestVoca/

Sem build, sem Vite. O app é 100% estático em docs/index.html.
